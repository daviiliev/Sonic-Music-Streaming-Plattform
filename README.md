# 🎵 SONIC – Music Streaming Landing Page

A premium, fully responsive landing page for a music streaming platform. Built with **Vanilla HTML, CSS & JavaScript**.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎵 **Interactive Player** | Play/pause with visual feedback |
| 📀 **Single-Slide Carousel** | Album showcase with dots & swipe |
| 📊 **Live Counter** | Real-time listener count |
| 🌊 **Animated Equalizer** | Moving wave bars |
| 📈 **Statistics Cards** | Animated counter on scroll |
| 🎨 **Teal & Mustard Palette** | Premium, balanced design |
| 📱 **Fully Responsive** | Perfect on all devices |
| ✨ **Scroll-Reveal** | Smooth animations on scroll |
